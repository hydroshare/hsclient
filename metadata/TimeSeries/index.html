A class used to represent the metadata associated with a time series aggregation

A time series aggregation consists of one or more time series datasets to which aggregation-level metadata have been added. Time series datasets in HydroShare consist of sequences of individual data values that are ordered in time to record the changing trend of a certain phenomenon. They are stored in HydroShare using ODM2 SQLite database files.

abstract: str pydantic-field

A string containing a summary of a resource

time_series_results: TimeSeriesResult pydantic-field required

A list of time series results contained within the time series aggregation

type: AggregationType pydantic-field

A string expressing the aggregation type from the list of HydroShare aggregation types

additional_metadata: dict pydantic-field

A list of extended metadata elements expressed as key-value pairs

language: str pydantic-field

The 3-character string for the language in which the metadata and content are expressed

period_coverage: PeriodCoverage pydantic-field

An object containing the temporal coverage for a resource expressed as a date range

rights: Rights pydantic-field

An object containing information about the rights held in and over the resource and the license under which a resource is shared

spatial_coverage: Union[hsmodels.schemas.fields.PointCoverage, hsmodels.schemas.fields.BoxCoverage] pydantic-field

An object containing the geospatial coverage for the resource expressed as either a bounding box or point

subjects: str pydantic-field

A list of keyword strings expressing the topic of the resource

title: str pydantic-field required

A string containing a descriptive title for the resource

url: AnyUrl pydantic-field required

An object containing the URL of the resource
